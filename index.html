Mazex — Maze Solver Robot

1st place · IEEE SB GECT Maze Solver Competition · GEC Thrissur

Introduction

Team Mazex participated in the Maze Solver Competition, organized by IEEE SB GECT at Government Engineering College (GEC), Thrissur. Competing against teams from across Kerala, our robot successfully navigated every section of the maze, securing 1st place in the competition.

This website documents our journey, including the design, testing, and development throughout the project.

// 3. Smooth "Soft Deadband" PD Calculation
float raw_error = filtered_L - filtered_R;
float error = 0.0;

// Instead of jumping instantly, the error ramps up
// smoothly once outside the deadband
if (raw_error > DEADBAND) {
  error = raw_error - DEADBAND;
} else if (raw_error < -DEADBAND) {
  error = raw_error + DEADBAND;
}

// Quicker derivative response to catch fast drifts earlier
float raw_derivative = error - last_error;
derivative = (0.4 * raw_derivative) + (0.6 * derivative);
last_error = error;

float steering_correction = (KP * error) + (KD * derivative);
